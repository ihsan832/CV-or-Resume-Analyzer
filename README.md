# 🧠 AI CV Analyzer (FastAPI + Gemini 2.5 Flash)

An AI-powered CV/Resume Analyzer built with **FastAPI** and integrated with **Google Gemini 2.5 Flash** model.  
This project automatically analyzes resumes, extracts key information, and provides intelligent feedback for improvement.

---

## 🚀 Features

- 📄 Upload and analyze CV/Resume (PDF/Text)
- 🤖 AI-powered evaluation using **Gemini 2.5 Flash**
- 🧠 Extracts:
  - Skills
  - Experience
  - Education
  - Projects
- 📊 Gives structured feedback for improvement
- ⚡ FastAPI backend for high performance API
- 🔌 Easy integration with frontend or other systems

---

## 🏗️ Tech Stack

- Python 🐍
- FastAPI ⚡
- Google Gemini 2.5 Flash 🤖
- LangChain / Prompt Engineering (if used)
- PyPDF2 / pdfminer (for PDF parsing)
- Uvicorn (server)
